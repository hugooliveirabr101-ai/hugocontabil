# Portfólio de Hugo Oliveira — versão final

Site pessoal responsivo desenvolvido em HTML, CSS e JavaScript, preparado para publicação no GitHub Pages.

## Como visualizar corretamente

Não mova o `index.html` para fora da pasta do projeto, pois ele utiliza os arquivos das pastas `css`, `js` e `assets`.

1. Extraia o arquivo ZIP completo.
2. Abra a pasta extraída.
3. Clique duas vezes em `index.html`.

## Estrutura

```text
meu-portfolio-final/
├── index.html
├── privacidade.html
├── favicon.svg
├── robots.txt
├── sitemap.xml
├── css/
│   └── style.css
├── js/
│   └── script.js
└── assets/
    ├── imagens/
    └── icones/
```

## Publicar no GitHub Pages

1. Crie um repositório público no GitHub.
2. Envie todos os arquivos e pastas do projeto, deixando `index.html` na raiz do repositório.
3. Abra **Settings**.
4. Entre em **Pages**.
5. Em **Source**, escolha **Deploy from a branch**.
6. Selecione a branch `main` e a pasta `/ (root)`.
7. Clique em **Save**.
8. Aguarde alguns minutos até o GitHub gerar o endereço do site.

## Ajuste depois da publicação

No arquivo `index.html`, substitua `SEU-USUARIO` na URL canônica pelo nome real da sua conta do GitHub. Faça o mesmo nos arquivos `robots.txt` e `sitemap.xml`.
